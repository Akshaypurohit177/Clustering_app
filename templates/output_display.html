Behind the scenes K means is trying to make K number of clusters of colors provided by you For eg:-if Algo has identified green color cluster then all the variations of green color are replaced by their cluster center or mean green color.
  1. Firstly create k clusters.
  2. then give cluster_label to each pixel (say '1' label to first pixel 'k' label to second pixel depending on which cluster that pixel belongs to)
  3. then label of that particular pixel is getting replaced by the actual value of its cluster center (or mean value of that color cluster) which ulitmately creates the effect
  4. In simple terms if we selected 2 clusters and K means identified them as say Green and Yellow, then all the pixels will be assigned to either mean green value(or Green cluster center) or mean yellow value(or Green cluster center)
  5. Also, point to be noted that each cluster center is an array containing 3 values of RGB (3 channels)
  1. flatten your image for fitting k means: X = img.reshape(-1, img.shape[2])
  2. Scale your image: X = X / 255
  3. Fit clustering model : kmeans = KMeans(n_clusters=k).fit(X)
  4. Assign cluster center values to all respective labels: segmented_img = kmeans.cluster_centers_[kmeans.labels_]
  5. reshape image to original size : segmented_img = segmented_img.reshape(img.shape)
